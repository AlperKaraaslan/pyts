class pyts.transformation.MTF(image_size=32, n_bins=8, quantiles=u'empirical', overlapping=False)[source]¶

Markov Transition Field.

Parameters:
image_size : int (default = 32)

determines the shapes of the output image : image_size x image_size

n_bins : int (default = 8)

number of bins (also known as the size of the alphabet)

quantiles : str (default = ‘gaussian’)
the way to compute quantiles. Possible values:
  • ‘gaussian’ : quantiles from a gaussian distribution N(0,1)
  • ‘empirical’ : empirical quantiles
overlapping : bool (default = False)

if False, reducing the image with the blurring kernel will be applied on non-overlapping rectangles; if True, it will be applied on possible overlapping squares.
